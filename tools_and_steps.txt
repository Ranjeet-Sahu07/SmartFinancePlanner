### Step 1: Clone the Repository
```bash
git clone https://github.com/Ranjeet-Sahu07/SmartFinancePlanner.git
cd SmartFinancePlanner
```

### Step 2: Verify Java Installation
```bash
java -version
# Should display: java version "17" or higher
```

### Step 3: Verify Maven Installation
```bash
mvn -version
# Should display Maven version 3.6+ and Java version
```

### Step 4: Download Dependencies
```bash
mvn clean install
# This downloads all required JavaFX libraries and plugins
```

### Step 5: Compile the Project
```bash
mvn clean compile
# Compiles all Java source files
```

### Step 6: Run the Application
```bash
mvn exec:java
# Launches the SmartFinancePlanner application
```

Alternatively, use the JavaFX Maven Plugin:
```bash
mvn javafx:run
```

## ğŸ› TROUBLESHOOTING

### Issue: "Error: JavaFX runtime components are missing"
**Solution:**
- Ensure you're using mvn exec:java or mvn javafx:run
- Do NOT use java -jar directly without JavaFX modules

### Issue: "Maven dependencies not downloading"
**Solution:**
```bash
mvn clean install -U
# -U forces update of dependencies
```

### Issue: "Java version mismatch"
**Solution:**
- Verify JAVA_HOME environment variable points to JDK 17+
- Update pom.xml if using different Java version

### Issue: "JavaFX CSS not loading"
**Solution:**
- Ensure style.css is in src/main/resources/styles/
- Check FXML files have correct CSS path references
